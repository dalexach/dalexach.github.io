# dalexach.github.io

Personal portfolio site for Daniela Chamorro — Python Backend & Data Engineer.

Live at: **[dalexach.github.io](https://dalexach.github.io)**

## Stack

- Plain HTML + CSS (no frameworks, no build step)
- [DM Serif Display / DM Mono / DM Sans](https://fonts.google.com/) via Google Fonts
- Hosted on GitHub Pages

## Structure

```
├── index.html          # Main page
├── assets/
│   └── css/
│       └── style.css   # All styles
└── images/
    └── Daniela.jpg     # Profile avatar
```

## Sections

- **Hero** — intro, stats
- **Skills** — languages, frameworks, cloud, AI/NLP tools
- **Experience** — work history (2020 – present)
- **Education** — degrees and programs
- **Projects** — personal GitHub projects
- **Contact** — email, phone, LinkedIn, WhatsApp

## Local development

No build tools needed — open `index.html` directly in a browser or use any static server:

```bash
npx serve .
# or
python3 -m http.server
```
